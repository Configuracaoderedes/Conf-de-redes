Estrutura:

Arquivos >> index.php, menu.php, app.php e css.css.

Diretório >> css

#############

Funcionalidades: 

* A aplicação permite, detectar os ativos da rede que estão disponíveis no momento, informando um range de IP desejado.

* É possível verificar a configuração de rede utilizada por cada dispositivo individualmente.

* Pode-se testar a conctivadade do dispositivo com a rede, através de PING.


#############

Configuração:

Servidor:

* Arquitetura de servidor Web tipo LAMP >> Sistema operacional Debian, Servidor web Apache, banco de dados MySQL e linguagem PHP.

* Servidor SSH ativo.

Cliente:

* Máquinas com arquitetura linux, com serviço SSH ativo.
